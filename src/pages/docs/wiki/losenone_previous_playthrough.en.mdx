## 注意事项

由于存档文件对于 Minecraft 基岩版来说体积过大，所以您可能会遇到游戏无法正常导入。\
故您导入存档时，请将存档文件（losenone&#95;&#42;&#42;.7z）解压至游戏存档存储目录下，重启游戏即可正常访问世界。

Minecraft 基岩版各分支版本存档存储目录：
1. Windows（正式版）
```
%USERPROFILE%\AppData\Local\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang\minecraftWorlds
```
2. Windows（预览版）
```
%USERPROFILE%\AppData\Local\Packages\Microsoft.MinecraftWindowsBeta_8wekyb3d8bbwe\LocalState\games\com.mojang\minecraftWorlds
```
3. Android（文档存储位置：外部）
```
/storage/emulated/0/Android/data/com.mojang.minecraftpe/files/games/com.mojang/minecraftWorlds
```
4. iOS/iPadOS/macOS
```
在文件里搜 minecraft 即可找到，若不会可自行搜索教程。
```
