# 归落原 · 白名玩家名单 / Minecraft


### 注意事项

1. 我们采用 UUID（通用唯一识别码）来储存玩家信息以及审核状态，如果您需要查询白名单权限审核状态请先准备好您的 UUID 。
2. 您可在 [mcuuid.net](https://mcuuid.net/) 查询到属于您的 UUID （前提是您拥有 Minecraft 正版并设置了 Minecraft ID），复制其给出的 Full UUID 项即可，它看起来长这样：
`00000000-0000-0000-0000-0000000000` 。
