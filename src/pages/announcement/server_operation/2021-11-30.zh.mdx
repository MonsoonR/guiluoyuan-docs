# 归落原运营调整公告 / 2021-11-30

<br />
亲爱的社区成员们，目前归落原开放的服务器周目均已圆满完结，感谢大家一直以来的支持与游玩！

归落原全新周目即将到来！

★ 全新基于 Minecraft: Java Edition 1.18 的纯净生存服务器，归落原 · 原初系列「第一周目」&#126;\(≧▽≦)/&#126;

归落原 · 原初系列「第一周目」会在我们内部调整与测试完善后上线开放，具体开放日期敬请期待！

归落原欢迎志同道合的人们加入我们这个大家庭，如果你是萌新玩家也不用慌张，您可向我们的疑问解答专员询问，同时我们会在归落原官网刊登游玩教程，以及开放已完成配置的整合包下载。

原初系列「第一周目」无需 Minecraft: Java Edition 正版账户也可游玩，若使用正版账户，归落原将会自动使用您正版账户的皮肤披风数据，非正版账户可在 [洛书南皮肤站](https://my.losenone.cn/) 注册您的游戏 ID 并配置皮肤披风数据。

原初系列「第一周目」依旧兼容玩家使用 Minecraft: Bedrock Edition 客户端登陆游戏，但开放初期这项服务可能不受支持，您可通过使用 PojavLauncher 等在手机系统上可用的 Minecraft: Java Edition 客户端进行登陆与游玩，我们会在归落原官网刊登使用教程。

游玩原初系列「第一周目」需先获取白名单权限，请持续关注公告，白名单权限申请通道会在后续公告中放出。
